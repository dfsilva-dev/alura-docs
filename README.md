# Alura Docs

## Descrição

Aplicação criada utilizando Node.js e Express para desenvolvimento de uma página web que possibilita, que possibilita aos usuários gerenciar documentos. Os documentos funcionam como blocos te textos que podem ser acessados  e alterados simultaneamente por mais de uma usuário, com o intuito de possibilitar o compartilhamento de anotações de estudos.

Os documentos são persistidos utilizando o MongoDB e a biblioteca de Socket.IO para comunicação entre clientes e servidor.

## Description

This application, created using Node.js and Express, develops a web page that allows users to manage documents. Documents function as blocks of text that can be accessed and modified simultaneously by more than one user, enabling the sharing of study notes.

Documents are persisted using MongoDB and the Socket.IO library for client-server communication.
